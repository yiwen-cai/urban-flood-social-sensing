### 幻灯片标题

### 左侧对比表（逐字）

| 维度 | tfidf-lr-baseline-v1 | deepseek-v4-flash |
|------|----------------------|-------------------|
| 类型 | TF-IDF + 逻辑回归 | LLM Few-shot 分类 |
| 覆盖率 | 100% | 90.5%（150 条失败） |
| 优势直觉 | 稳定、可复现、零 API | 语义边界与稀缺类更灵活 |
| 风险 | 易偏向高频「救援志愿」类 | 成本/失败样本需处理 |
| 课程演示 | 始终可离线重跑 | 复用已跑批预测，演示可不调 API |

### 右侧三条理由（逐字）

1. **表述多样**：灾害推文同义表达多，纯词袋易被多数类淹没。  
2. **决策相关类**：下一页显示 DeepSeek 在 `requests_or_urgent_needs` 上 Recall **70.5%**，高于 baseline 的 **41.0%**；Macro-F1 **0.553 vs 0.403**。  
3. **可交付**：预测已落盘；`bash run_pipeline.sh` 即可重建看板，不依赖现场密钥。
